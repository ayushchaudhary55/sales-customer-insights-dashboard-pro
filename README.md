# 📊 Sales & Customer Insights Dashboard Pro

A Business Analyst portfolio project built using Python, Streamlit, SQL-ready analytics, Excel/CSV support, Power BI-ready exports, and AI-style business recommendations.

This project analyzes sales and customer datasets to identify revenue trends, product performance, regional insights, customer behavior, and operational KPIs through an interactive dashboard.

---

# 🚀 Features

* Interactive Streamlit dashboard
* Built-in sample dataset with 350+ business records
* CSV and Excel upload support
* SQLite database integration
* SQL business analysis queries
* Power BI-ready export files
* KPI tracking and reporting
* Monthly sales trend analysis
* Product & category performance analysis
* Regional sales distribution analysis
* Payment mode analysis
* Top customers & product insights
* AI-style business recommendations
* Downloadable filtered reports

---
