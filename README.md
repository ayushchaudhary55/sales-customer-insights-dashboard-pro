# Sales & Customer Insights Dashboard Pro

A Business Analyst portfolio project built with Python, Streamlit, SQL-ready data, Excel/CSV support, Power BI-ready exports, and AI-style business recommendations.

## Features
- Interactive Streamlit dashboard
- Built-in sample dataset with 350+ orders
- Upload your own CSV or Excel file
- SQLite database integration
- SQL analysis queries included
- Power BI-ready CSV exports
- KPI tracking and reporting
- Monthly sales trend analysis
- Product/category performance analysis
- Regional sales share analysis
- Payment mode analysis
- Top customer analysis
- AI-style business recommendations
- Downloadable filtered report

## Tech Stack
Python, Streamlit, Pandas, Plotly, SQLite, Excel/CSV, Power BI-ready exports

## How to Run
```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

## Required Upload Columns
Order_ID, Order_Date, Customer_ID, Customer_Name, Region, State, City, Product_Category, Product_Name, Quantity, Unit_Price, Discount, Sales, Cost, Profit, Payment_Mode
